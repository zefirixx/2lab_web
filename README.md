## Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd html-form-lab
Запустить контейнеры:
```bash
docker-compose up -d --build
```
Открыть в браузере:
```http://localhost:8080```

Содержимое проекта:

```docker-compose.yml``` — описание сервиса Nginx

```www/index.html``` — главная HTML-страница

```www/index.php``` — страница с подробной информацией о PHP 

```default.conf``` — конфиг c добавленной секцией для обработки PHP

```screenshots/``` — все скриншоты

Результат:
Nginx отдаёт мою HTML-форму по теме "Регистрация на конференцию".
